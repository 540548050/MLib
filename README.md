# react-boilerplate

大数金科项目前端初始化模版

## 说明

## 包含功能

- react-scripts
- mobx
- react-router
- typescript
- http-proxy-middleware
- sass
- css-module
- jest

### 浏览器支持

IE 9+

### 运行

dev

```sh
yarn start
```

production

```sh
yarn build

serve ./build
```

### 开发设置

1. setupProxy.js 修改后端代理
1. 运行时修改后端 apiServer http://localhost:3000//settings?apiServer=https://trade.zsteel.cc
